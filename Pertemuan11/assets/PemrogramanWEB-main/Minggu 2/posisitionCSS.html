position: static

Position static digunakan untuk mengatur elemen menjadi static menjadi statis secara default. Elemen dengan posisi static tidak memiliki properti top, right, bottom, dan left yang dapat diubah oleh pengguna.:

Elemen ini diatur menggunakan posisi: static;




position: relative

Sebuah elemen html yang menggunakan position relativ akan terletak pada posisi normal. Mengatur porperti atas, kanan, bawah, dan kiri dari elemen yang diposisikan dengan "relatif" akan membuatya jauh dari posisi normalnya Konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut.:

Elemen ini diatur menggunakan posisi : relative;



position: fixed;

Sebuah elemen html yang disetting menggunakan position fixwd akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scroll. Berlaku pengaturan left, bottom, top, and right pada position fixed:

Elemen ini diatur menggunakan position:fixed;















position: absolute;

Elemen HTML yang memiliki posisi absolut akan bergerak sesuai dengan properti-properinya. bukan relatif terhadap layar secara normal.:

Elemen yang diatur menggunakan posisi: relative;
Elemen ini diatur menggunakan posisi: absolute;
