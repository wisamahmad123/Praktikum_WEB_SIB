Ini adalah contoh Link Relatif

Klik di sini Jika Anda ingin menuju ke halaman berikutnya.

Link relatif adalah jenis tautan dalam HTML yang merujuk ke halaman atau sumber daya lain dalam satu situs web atau komputer yang sama. Contohnya, jika Anda memiliki halaman "index.html" dan ingin menuju ke halaman "about.html" yang berada dalam folder yang sama, Anda dapat menggunakan link relatif seperti About.

Ini adalah contoh Link Absolut

Klik di sini untuk pencarian yang Anda inginkan.

Link absolut merujuk ke suatu halaman atau sumber daya dengan menggunakan alamat lengkap atau URL. Ini bisa mencakup domain, subdomain, path lengkap, dan nama berkas. Sebagai contoh, About.

Ini adalah contoh Link dengan Window Baru

Buka www.google.com di halaman baru.

Link dengan jendela baru digunakan untuk membuka tautan dalam jendela atau tab baru. Ini dicapai dengan menambahkan atribut target="_blank" pada elemen tautan. Misalnya, Link Baru.
